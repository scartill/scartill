## 📦 PyPI Packages

### [easysam](https://github.com/scartill/easysam) · [![PyPI](https://img.shields.io/pypi/v/easysam)](https://pypi.org/project/easysam/)
Opinionated YAML-to-SAM generator for modular AWS serverless applications.
Define Lambda functions, API Gateway routes, DynamoDB tables, S3 buckets, SQS queues, Kinesis streams, OpenSearch Serverless collections, and IoT Core authorizers in a compact `resources.yaml` — then generate and deploy the SAM stack with a single command.

### [prismarine](https://github.com/scartill/prismarine) · [![PyPI](https://img.shields.io/pypi/v/prismarine)](https://pypi.org/project/prismarine/)
Pythonic DynamoDB ORM. Define models as `TypedDict` (or Pydantic) classes, auto-generate a type-safe client with full CRUD methods, and integrate seamlessly with EasySAM-managed tables.

### [tagth](https://github.com/scartill/tagth) · [![PyPI](https://img.shields.io/pypi/v/tagth)](https://pypi.org/project/tagth/)
Stateless tag-based authorization library. A lightweight alternative to RBAC: map principal tags and resource tags to a binary allow/deny decision — no persistence required.

### [botobuddy](https://github.com/scartill/botobuddy) · [![PyPI](https://img.shields.io/pypi/v/botobuddy)](https://pypi.org/project/botobuddy/)
A collection of utility functions and CLI commands built on top of Boto3.
Covers DynamoDB, S3, Route 53, SageMaker, API Gateway, Lambda, and Secrets Manager helpers.

### [dagster-static](https://github.com/scartill/dagster-static) · [![PyPI](https://img.shields.io/pypi/v/dagster-static)](https://pypi.org/project/dagster-static/)
Trait-based static validation system for Dagster assets. Catch dependency mismatches at definition time rather than at runtime, with clear error messages and namespace support.
